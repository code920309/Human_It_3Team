# CareLink 서비스 기획서 (PRD)

## 1. 제품 개요 (Product Overview)
CareLink는 AI 기술을 활용하여 복잡한 건강검진 결과표를 한눈에 이해하기 쉬운 리포트로 변환하고, 사용자의 건강 데이터를 지속적으로 관리할 수 있도록 돕는 개인 맞춤형 헬스케어 플랫폼입니다.

## 2. 타겟 사용자 (Target Audience)
- 건강검진 결과표의 전문 용어와 수치를 이해하기 어려운 일반인
- 과거 건강검진 기록을 한곳에 모아 변화를 추적하고 싶은 사용자
- 일상 속에서 맞춤형 건강 관리 가이드(식단, 운동 등)를 받고 싶은 사람

## 3. 핵심 기능 (Key Features)

### 3.1 AI 건강 분석 (OCR & Analysis)
- **이미지 업로드**: 건강검진 결과표 사진을 업로드하면 AI가 텍스트를 인식(OCR)합니다.
- **지표 추출**: 혈당, 콜레스테롤, 간 수치 등 주요 건강 지표를 자동으로 추출합니다.
- **AI 요약 및 스코어링**: 전체적인 건강 상태를 요약하고 100점 만점의 'Health Score'를 산출합니다.

### 3.2 리포트 관리 및 수동 수정
- **지표 수동 기입/수정**: AI가 인식하지 못한 항목을 직접 추가하거나 수정할 수 있습니다.
- **자동 상태 판별**: 입력된 수치와 참고 범위를 비교하여 **정상(초록)**, **주의(노란)**, **위험(빨간)** 상태를 자동으로 분류합니다.
- **맞춤형 액션 플랜**: 분석 결과에 따른 식단, 운동 가이드, 생활 습관 개선안을 제공합니다.

### 3.3 사용자 경험 (UX) 최적화
- **비회원 경험**: 로그인 없이도 리포트 분석 기능을 즉시 이용할 수 있습니다.
- **데이터 심리스 마이그레이션**: 비회원 상태에서 생성된 리포트는 회원가입 즉시 해당 계정으로 자동 저장되어 유실되지 않습니다.
- **발표 모드**: 의사 상담이나 가족 공유 시 활용할 수 있는 깔끔한 전체 화면 모드를 제공합니다.

### 3.4 대시보드 및 AI 챗봇
- **건강 추이 차트**: 과거 리포트 데이터를 기반으로 건강 점수 및 주요 지표의 변화를 시각화합니다.
- **AI 챗봇 (CareBot)**: 자신의 건강 데이터를 바탕으로 궁금한 점을 묻고 전문적인 조언을 받을 수 있습니다.

## 4. 사용자 흐름 (User Flow)
1. **랜딩 페이지**: 서비스 소개 확인 및 분석 시작.
2. **업로드 및 분석**: 검진표 이미지 업로드 -> AI 분석 대기.
3. **리포트 확인**: 분석 결과 확인 및 필요 시 지표 수동 수정.
4. **회원가입 유도**: 리포트 하단 안내를 통해 가입 페이지 이동.
5. **데이터 저장**: 가입 완료 시 분석 중이던 리포트가 자동으로 마이페이지에 저장.
6. **사후 관리**: 마이페이지 대시보드 확인 및 챗봇 상담.

## 5. 기술 스택 (Technical Requirements)
- **Frontend**: React, Tailwind CSS, Framer Motion, Lucide React
- **Backend/DB**: Firebase (Authentication, Firestore)
- **AI**: Google Gemini API (Analysis & Chatbot)
- **Charts**: Recharts

## 6. 향후 로드맵 (Future Roadmap)
- 가족 계정 연결 및 건강 데이터 공유 기능
- 웨어러블 기기(Apple Health, Google Fit) 데이터 연동
- 정기 건강검진 예약 알림 및 병원 추천 시스템
